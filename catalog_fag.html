Установка серверного и клиентского ПО SL NEO, настройки ОС Windows

Configure Server Components – консоль конфигурации базовых аппаратных и программных компонентов сервера SL NEO. Консоль позволяет сформировать "набор" аппаратных узлов и серверных программных модулей - задать модели и количество плат ввода-вывода, количество каналов записи/воспроизведения сервера.

Важно ! Настройки серверной платформы должны производиться квалифицированными специалистами, которые прошли обучение и имеют соответствующие полномочия. Следует помнить, что неквалифицированные действия могут повлиять на работоспособность серверной платформы, при добавлении новых компонентов текущая лицензия может оказаться недействительной.

Свяжитесь со службой поддержки для уточнения деталей ! Процедура выбора базовых модулей производится при сборке сервера изготовителем, в соответствии с заказанной конфигурацией, тем не менее, данная процедура будет описана ниже.

Консоль запускается при остановленном серверном ПО  SL NEO (после выполнения команды Stop Server, по правой кнопке мыши на красном значке загрузчика SkyLark NeoVid в панели задач Windows). Настройки производятся через диалоговое окно, содержащее 2 основных окна Available components и Installed Components.

Video IO Boards – раздел содержит список поддерживаемых плат ввода-вывода.
LAN Interfaces – интерфейс для организации каналов записи/воспроизведения с использованием IP-технологий (протоколы UDP/RTP).
FireWire Interfaces – интерфейс для организации каналов записи/воспроизведения с использованием интерфейса IEEE1394 (DV25/HDV).
Computer Screen – интерфейс для организации канала воспроизведения с использованием встроенного VGA-адаптера сервера.

Software components - базовые серверные программные модули платформы SL NEO:
Program Channel - модуль автоматизированного исполнения рlay-листа для одного канала (сервис SL NEO Program Player).
Recorder - модуль одноканального рекордера.
Time Shift – модуль, выполняющий функции временной задержки от 3-х минут до
